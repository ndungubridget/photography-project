# Bridget Photography Portfolio

A modern, responsive photography portfolio website built using HTML5 and CSS3. The website showcases photography services, featured work, and contact information through an elegant and professional design.

## Live Preview

Open `index.html` in your web browser to view the website.

## Features

- Responsive design for desktop, tablet, and mobile
- Modern hero sections with background images
- Professional navigation bar
- About page with photographer profile
- Photography services with pricing
- Image gallery with hover effects
- Contact form
- Client testimonials
- Call to action sections
- Smooth hover animations
- Clean and elegant user interface

## Installation

1. Download or clone the project.
2. Open the project folder in Visual Studio Code.
3. Place all required images inside the `images` folder.
4. Open `index.html` in your browser.
